# Digitalizacion-Juego-de-Memoria-Pokemon-90s
Este proyecto esta enfocado en digitalizar y remasterizar un juego de memoria de Pokemon de los 90s en React.
![about2](https://github.com/user-attachments/assets/4910e1f1-2e12-4486-87a3-ba5073345341)
<img width="1914" height="900" alt="homescreen" src="https://github.com/user-attachments/assets/caa9b43c-dc91-47b1-a874-4c01787693b5" />
Empecé este proyecto para practicar React y mejorar mis habilidades de programación. Me regalaron este juego de memoria de Pokémon cuando tenía unos 8 años y quería revivirlo en la era digital de una forma más moderna y divertida.

Este juego de mesa se creó al principio de la era Pokémon, así que las traducciones de los nombres (en este caso al español) no estaban muy bien establecidas. Pero esta es la gracia de este proyecto, mantenerlo tan auténtico como cuando salió en los 90s. Al emparejar dos cartas, añadí una llamada a un servidor API para recuperar los datos del Pokémon y mostrarlos como una Pokédex con los sonidos originales de los juegos de la Game Boy para cada Pokémon.

Cada carta Pokemon fue digitalizada y corregida de defectos fisicos.

<img width="1231" height="619" alt="cards" src="https://github.com/user-attachments/assets/1562ff61-88ac-4bba-ba91-e03fa2dc525b" />

El juego cuenta con un reproductor de musica con pistas relacionadas al universo Pokemon. El juego cuenta tambien con diferentes temas con paletas de colores relacionados a los principales pokemones de la saga (Pikachu, Charmander, Bulbasaur, Squirtle, Mewtwo).

Se añadio un sistema de cara o sello para ver quien empieza la partida.

# Digitization-Memory-Game-Pokemon-90s
This project focuses on digitizing and remastering a 90s Pokemon memory game in React.
![about2](https://github.com/user-attachments/assets/4910e1f1-2e12-4486-87a3-ba5073345341)
<img width="1914" height="900" alt="homescreen" src="https://github.com/user-attachments/assets/caa9b43c-dc91-47b1-a874-4c01787693b5" />
I started this project as a way to practice React and improve my coding skills. I got this Pokemon memory board game when I was around 8 years old and I wanted to bring it back to life to the digital era in a more modern and fun way.

This board game was made at the start of the Pokemon era, so translated names (in this case Spanish) were not that polished. But that's the beauty of it, to keep it as authentic as it was when it came out. When you match a pair of cards, I added a call to an API server to retrieve data of the matched Pokemon and display it as a Pokedex with the original Game Boy games sounds for each Pokemon.

Each Pokemon card was digitized and edited for physical defects.

<img width="1231" height="619" alt="cards" src="https://github.com/user-attachments/assets/1562ff61-88ac-4bba-ba91-e03fa2dc525b" />

The game features a music player with tracks related to the Pokémon universe. The game also features different themes with color palettes inspired by the series' main Pokémon (Pikachu, Charmander, Bulbasaur, Squirtle, Mewtwo).

A heads or tails system was added to see who starts the game.

<img width="1919" height="901" alt="Pokedex" src="https://github.com/user-attachments/assets/f8edc72f-1ffc-4027-9173-7e5f66ab23fc" />

The game has three difficulty levels for the CPU: Easy, Normal, and Hard. I haven't been able to beat Hard mode, so I wish you luck! :)

Controls:
- Space Bar: Play/Pause music
- N: Next track
- P: Previous track
- M: Mute/Unmute
- +: Volume up
- -: Volume down
- T: Change theme
